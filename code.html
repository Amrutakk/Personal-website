Python Examples

Example 1: Print right-angle triangle 
for i in range(5):
  for j in range(i+1):
    print("*", end=" ")
  print()
Nested loops print increasing number of stars.
Output:
*
* *
* * *
* * * *
* * * * *
Example 2: Fibonacci series 
a, b = 0, 1
for _ in range(5):
  print(a, end=" ")
  a, b = b, a + b
Prints first 5 Fibonacci numbers using loop & swapping.
Output: 0 1 1 2 3
